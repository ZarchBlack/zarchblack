## 🎯 About ZarchBlack

**ZarchBlack** is a cutting-edge Linux distribution built on top of **Arch Linux**, meticulously designed for:

- 🧑‍💻 **Software Developers** - Complete development environment
- 💻 **Programmers** - Multi-language support and IDEs
- 🔐 **Cybersecurity Professionals** - Advanced security tools
- 🎯 **Ethical Hackers** - Penetration testing frameworks
- ⚙️ **System Administrators** - Advanced system management tools
- 🚀 **Advanced Linux Users** - Full customization and control

ZarchBlack combines the power and flexibility of Arch Linux with carefully curated specialized tools, providing an **out-of-the-box professional environment** for security research, software development, and system administration.

## 🚀 Installation

### Prerequisites

- USB drive (8 GB minimum)
- Computer with 30 GB free disk space
- BIOS/UEFI boot capability
- Internet connection for post-installation setup

### Step 1: Download ZarchBlack

Download the latest ISO from:
```
https://huggingface.co/datasets/zarchblack/zarchblack-releases/tree/main
```

### Step 2: Create Bootable USB

**Using Balena Etcher (Recommended - Cross-platform):**
1. Download [Balena Etcher](https://www.balena.io/etcher/)
2. Open Balena Etcher
3. Select the ZarchBlack ISO file
4. Select your USB drive
5. Click "Flash"

**Using `dd` (Linux/macOS):**
```bash
# Identify your USB device (e.g., /dev/sdb)
lsblk

# Unmount the USB device
sudo umount /dev/sdX*

# Write the ISO
sudo dd if=zarchblack-v1.0.iso of=/dev/sdX bs=4M status=progress
sudo sync
```

**Using Rufus (Windows):**
1. Download [Rufus](https://rufus.ie/)
2. Select the ZarchBlack ISO
3. Configure settings (default is fine)
4. Click "Start"

### Step 3: Boot from USB

1. Insert the USB drive into your computer
2. Restart your computer
3. Enter BIOS/UEFI setup (usually F2, F10, Del, or Esc during startup)
4. Set USB device as the primary boot device
5. Save and exit
6. Boot into ZarchBlack installer

### Step 4: Run Installer

1. The ZarchBlack installer will load
2. Follow the on-screen installation wizard:
   - Select language and keyboard layout
   - Choose installation disk
   - Configure partitioning (auto or manual)
   - Set hostname
   - Create user account and password
   - Select additional packages/tools
3. Review installation summary
4. Click "Install" to begin installation
5. Wait for installation to complete (~10-15 minutes)

### Step 5: Post-Installation

1. Remove USB drive when prompted
2. Reboot the system
3. Log in with your credentials
4. Run initial setup wizard: `zarch-setup`
5. Configure your preferences and tools

## ⚙️ Quick Start

### First Boot - Initial Setup

After logging in for the first time, run the ZarchBlack setup wizard:

```bash
zarch-setup
```

This interactive wizard will guide you through:
- System optimization and hardening
- Tool configuration (ZarchHacking, ZarchGuard, ZPackageManager)
- Development environment setup
- Security settings
- Desktop customization

### Update Your System

Keep ZarchBlack current with security patches and latest packages:

```bash
# Update all packages
sudo pacman -Syu

# Or use ZarchGuard for guided updates
zarch-guard update
```

### Install Development Tools

ZarchBlack comes with popular IDEs, but you can install more:

```bash
# Using ZPackageManager
zpackagemanager install code        # VS Code
zpackagemanager install jetbrains-toolbox  # JetBrains
zpackagemanager install vim         # Vim

# Or using pacman directly
sudo pacman -S base-devel git docker
```

### Access Security Tools

Launch the hacking and security framework:

```bash
# Interactive interface
zarch-hacking

# CLI mode for specific tools
zarch-hacking --search nmap
zarch-hacking --category reconnaissance
```

### System Maintenance

Keep your system optimized and clean:

```bash
# Interactive maintenance
zarch-guard

# Specific operations
zarch-guard clean           # Deep clean
zarch-guard update          # Update system
zarch-guard check           # Check system health
zarch-guard repair          # Fix issues
```

### Install Packages

Manage software installations easily:

```bash
# Search for a package
zpackagemanager search python

# Install a package
zpackagemanager install python

# Remove a package
zpackagemanager remove package-name

# Update all packages
zpackagemanager update

# Manage repositories
zpackagemanager repo list
zpackagemanager repo add [repo]
```
